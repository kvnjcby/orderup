RESTful API

  • /createFood POST Parameters: name, description, price, cookTime. Adds food to database.
  • /getFood GET Parameters: id. Returns food matching id in JSON format.
  • /getFoods GET Parameters: N/A. Returns all foods in JSON format.
  • /deleteFood DELETE Parameters: id. Deletes food by id.
  • /createOrder POST Parameters: customerName, orderStatus, foods. Foods parameter is a JSON list of food id's. Creates new order.
  • /getOrder GET Parameters: id. Returns single order in JSON format.
  • /getOrders GET Parameters: (optional)orderStatus. Returns all orders in JSON format.
  • /updateOrder PUT Parameters: id, orderStatus. Updates the status of an order based on id.
  • /deleteOrder DELETE Parameters: id. Deletes an order by id.
